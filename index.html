post /user/login
DONE -- Logs user into the system (loginUser)
Login API sends user's credentials to cognito, in return, if valid credentials, cognito sends back token which could be use to call other API's. Also returns user details, servers list, countries list

In case of incorrect credentials, it returns
status:200
status_code:400,
message:Incorrect username or password.
code:NotAuthorizedException
time:Error time
In case of user does not exist, it returns
status:200
status_code:400,
message:User does not exist.
code:UserNotFoundException
time:Error time
